#Please read the following instruction before running the code.{

{first insall mysql.connector library using 'pip install mysql-connector-python' in windows cmd.}

1) Open the db_info.txt file and enter your database info.

	-Points to remember:

	a) Always right after '=' symbol.
	b) Remove by-default sample info wrote after '='.
	c) Replace the sample info by you database info.
	d) The overall file should look like;
	
	   host=<your database host name>
	   user=<your database user name>
	   password=<your database password>
	   database=<your database name>

	e) No spaces should be given after and before '='.
